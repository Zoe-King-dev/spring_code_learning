  C-307 详情（学习模式差异化设计）

  背景：见 B-08。当前 [阅读｜学习] tab 切换后无任何可感知差异。

  设计候选（择一或组合）：
    A. 增强阅读侧
       - 阅读模式：纯文字渲染，零干扰
       - 学习模式：每段后追加 "本段要点" 折叠卡 + 关键术语 📌 标记
    B. 嵌入测验侧
       - 阅读模式：默认
       - 学习模式：每章首屏后强制弹 3 题章节测验，做完才解锁后续
    C. 可视化侧
       - 阅读模式：Mermaid 默认折叠（仅占位）
       - 学习模式：Mermaid 全部展开 + 自动联动左侧知识图谱
    D. 复习侧
       - 学习模式：把当前章节的错题、笔记、收藏以侧栏抽屉聚合
         显示，提供 "复习上一节" 入口

  推荐组合：A + D。A 给正文加值、D 利用现有 Storage 数据零成本。

  验收：
    - 切换 [学习] 模式后 1 秒内出现 ≥2 个可感知差异
    - 不影响阅读模式表现
    - 用户可一键回到阅读模式
