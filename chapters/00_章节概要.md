# 章节概要

## 第1章：Spring Boot 整体概述

了解 Spring Framework 与 Spring Boot 的关系、核心特性以及体系结构，学习如何创建第一个 Spring Boot 应用。

## 第2章：Spring Boot 的自动装配

深入理解组件装配的概念，探索 Spring Framework 的模块装配、条件装配以及 SPI 机制，掌握 Spring Boot 自动装配的核心原理。

## 第3章：Spring Boot 的 IOC 容器

系统学习 Spring Framework 的 IOC 容器模型，包括 BeanFactory、ApplicationContext、Environment、BeanDefinition 以及各种后置处理器。

## 第4章：Spring Boot 的核心引导：SpringApplication

理解 Spring Boot 的核心引导类 SpringApplication 的设计和生命周期。

## 第5章：Spring Boot 的 AOP 支持

回顾 Spring Framework 的 AOP 模块，以及 Spring Boot 整合 AOP 后的关键开关和装配底层。

## 第6章：Spring Boot 准备容器与环境

完整理解 Spring Boot 应用启动流程中的 SpringApplication 部分，包括创建、启动关键逻辑以及不同版本的扩展。

## 第7章：IOC 容器的刷新

全方位讲解 ApplicationContext 的容器初始化全流程，对 Spring Framework 的 IOC 容器初始化逻辑进行深度剖析和原理解读。

## 第8章：Spring Boot 容器刷新扩展：嵌入式 Web 容器

深入了解 Spring Boot 的嵌入式 Web 容器模型，以嵌入式 Tomcat 为例讲解其核心工作流程。

## 第9章：AOP 模块的生命周期

从 AOP 模块的核心后置处理器切入，讲解 AOP 机制在收集增强器、代理 bean 对象的创建过程以及调用代理对象时内部的执行机制。

## 第10章：Spring Boot 整合 JDBC

讲解 Spring Boot 整合 JDBC 场景的自动装配，以及整合事务模块后的生效原理、控制原理和事务传播行为原理。

## 第11章：Spring Boot 整合 MyBatis

讲解 Spring Boot 整合 MyBatis 操控数据库的自动装配，以及 MyBatis 的核心组件 SqlSessionFactory 的初始化流程。

## 第12章：Spring Boot 整合 WebMvc

讲解 Spring Boot 整合 WebMvc 场景的自动装配，以及 WebMvc 中的核心组件和 DispatcherServlet 的工作全流程。

## 第13章：Spring Boot 整合 WebFlux

从响应式编程开始讲解 WebFlux 的基础和快速使用，讲解 Spring Boot 整合 WebFlux 场景的自动装配，以及核心控制器 DispatcherHandler 的工作全流程。

## 第14章：运行 Spring Boot 应用

通过不同的 Spring Boot 应用打包方式讲解应用的引导启动流程，并讲解 2.3 版本引入的优雅停机特性。
