内容提要

Spring Boot 是⽬前 Java EE 开发中颇受欢迎的框架之⼀。依托于底层 Spring Framework 的基础⽀

撑，以及完善强⼤的特性设计，Spring Boot 已成为业界流⾏的应⽤和微服务开发基础框架。

本书共14章，分为4个部分。第1部分介绍 Spring Boot 底层依赖的核⼼容器，以及底层 Spring

Framework 的两⼤核⼼特性IOC AOP；第2部分从底层源码⾓度深⼊剖析 Spring Boot 的全⽅位⽣

命周期，包括 SpringApplication、IOC容器、嵌⼊式Web 容器和AOP 模块的⽣命周期；第3部分针对

项⽬开发中整合的主流场景，介绍场景模块中的核⼼装配和关键机制原理，如JDBC中的事务、Web

中的核⼼控制器等；第4 部分与 Spring Boot 的运⾏部署相关，针对不同运⾏场景讲解 Spring Boot 的

阅读本书之前，读者需要先对 Spring Framework 和 Spring Boot 有基本的理解与简单的框架使⽤经

验或项⽬开发经验。本书的重点是 Spring Boot 的设计、思想和原理，⽆论是对于已经有⼀定基础的开

发者还是已熟练使⽤ Spring Boot 并希望进⼀步提升技能和⽔平的开发者，本书都是他们透彻研究

Spring Boot 源码和原理的理想选择。

Spring Boot 从诞⽣⾄今已有数个年头，依托其简单易⽤、覆盖场景⼴泛、满⾜分布式应⽤

快速开发等特性，迅速成为互联⽹软件开发的⾸选基础框架。Spring Boot 本⾝设计强⼤、巧妙，
